## Feature Description

An AI-powered e-commerce catalog management tool that retrieves catalog
data, optimizes and generates product or catalog descriptions and SEO
data leveraging a "brand voice". Uses Vercel Durable Agents for
long-running optimization workflows with per-item streaming progress.
Accessible and monitorable through unique URLs. Requires human approval
before finalizing and saving content changes.
